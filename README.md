# ~~AR~~ GEOLOCATION SNAKE!!!

## Usage

Install dependencies:

```bash
yarn global add react-native-cli
yarn
```

Fill `config.js` with:

```javascript
export const GOOGLE_MAPS_KEY = "";
```

Run packager:

```bash
react-native start
```

Run on Android or iOs:

```bash
react-native run-android
react-native run-ios
```
